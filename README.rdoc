== About

This is a simple rails application created to compare multiple javascript chart libraries (both free and paid).

Javascript graph libraries used:

* RGraph[http://www.rgraph.net/]
* FusionCharts[http://www.fusioncharts.com/]
* AMCHARTS[https://www.amcharts.com/]
* Highcharts[http://www.highcharts.com/]
* Dimple[https://github.com/PMSI-AlignAlytics/dimple]
* C3[https://github.com/masayuki0812/c3]
